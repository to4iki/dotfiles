# AGENTS.md - Global Level

## 開発スタイル

- TDDで開発する（探索 → Red → Green → Refactoring）
- 不明瞭な指示は質問して明確にする

## ワークフロー

軽微な修正以外の実装では、**`git-wt` スキル**で worktree を作成する。

- 適用条件:
  - Plan mode 終了後、コード変更を伴う実装を開始する時
  - 複数ファイルにまたがる変更（機能追加、リファクタ、設定の横断的な更新など）
- 除外条件:
  - 単一ファイルの軽微な修正（typo、コメント追加等）
  - README やドキュメントのみの変更
- 注意点:
  - `git worktree` は使わない
  - 手順・ブランチ名・クリーンアップはスキルに従う
  - 迷った場合は worktree で作業する
